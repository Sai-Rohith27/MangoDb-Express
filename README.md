MongoDB + Express Integration
This project demonstrates how to integrate **MongoDB** with an **Express.js** server using **Node.js**. It includes basic CRUD operations and a simple backend setup.
# Features
- Connect to MongoDB database
- Create (Insert) data
- Read (Fetch) data
- Update existing data
- Delete data
- REST API using Express
---
#Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
